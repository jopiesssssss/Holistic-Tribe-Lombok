Kuara gallery patch v4

Upload this patch into the root of your Holistic Tribe Lombok GitHub repository and overwrite existing files.

Included:
- facilitators/kuara-lombok-south-lombok.html
- facilitators/kuara-resort-lombok.html
- assets/images/kuara/gallery/thumbs/*.webp

What changed:
- Gallery images are shown with object-fit: contain so they are not heavily cropped.
- A soft blurred background fills empty space behind portrait photos.
- Separate lightweight thumbnails are used, so the page does not load all large gallery images as thumbnails.
- A lightbox has been added: click the large image or “View large” to open a bigger viewer.
- Arrow navigation and Escape-to-close are included.
