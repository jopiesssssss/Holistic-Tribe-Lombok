Kuara gallery patch v5

Upload this patch into the root of your Holistic Tribe Lombok GitHub repository and overwrite existing files.

This patch changes the Kuara page gallery to a Booking-style photo collage:
- one large image + four smaller images
- Show all photos button
- full gallery modal
- click any photo to view it larger
- separate lightweight thumbnails for the full gallery modal

You do not need patch 3 or patch 4 first. This patch can go directly over patch 2.
